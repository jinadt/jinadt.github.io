HTML Table Reference

Table Tags

Tag Name Description
<table> Table This contains all of the tabular data we plan on displaying.
<tr> Table Row The table row element is used to add rows to a table before data and table headings.
<td> Table Data The table data element can be nested inside a table row element to add a cell of data to be a table.
<thead> Table Head The tbale element defines the headings of table colums encapsulated in table rows.
<th> Table heading This is used to add titles to rows and columns of a table and must be enclosed in a table row element.
<tbody> Table Body Table Body contains all table data other than table heading and table footer content.
<tfoot> Table Footer The table footer uses table rows to give footer content or to summarize content at the end of a table.

Table Attributes

Attribute Name Description
colspan Column Span It defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

Fundamentals of Html

Tag Name Description
<li> List Item The list item element create list items inside Ordered lists <ol> and Unordered lists <ul>
<video> video The <video> element embeds a media player for video playback. The src attribute will contain the URL to the video. Adding the controls attribute will display video controls in the media player. Note: The content inside the opening and closing tag is shown as a fallback in browsers that don't support the element.
<em> Emphasis The <em> element emphasizes text and browsers will usually italicize the emphasized text by default.
<ol> Ordered List The <ol> ordered list element creates a list of items in squential order.Each list item appears numbered by default.
<div> Div Element This is used as a container that divides an HTML document into sections and is short for "division". it can contain flow content such as headings, paragraphs, links, images, etc.
<br> Line Break The <br> line break element will create a line break in text and is especially useful where division of text is required, like in a postal address.
<img> image Element The <img> element embed images in documnents. The src attribute contains the image URL and is mandatory. <img> is an empty element meaning it should not have a closing tag.
<h1>-<h6> Heading Elements HTML can use six different levels of heading elements. The heading elements are ordered from the highest level <h1> to the lowest level <h6>.
